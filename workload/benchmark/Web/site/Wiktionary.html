Alternative forms[edit]

Etymology[edit]

Blend of Linus +‎ Unix, from the name of its creator Linus Torvalds.

Pronunciation[edit]

  • IPA(key): /ˈlɪnəks/, /ˈlɪnʊks/

Proper noun[edit]

Linux (plural Linuxes)

  1. An open-source computer operating system kernel.
  2. Any operating system that uses this kernel.
